Introduce

Pudron is an easy-to-use, lightweight, extensible and high potential interpretative programming language

Pudron's advantages:

  • The format of code is pithy,wonderful and humanized
  • Support many programming methods,such as object-oriented and Process-oriented
  • Extensible,using dynamic link library to expand the Pudron library
  • High potential

Pudron's shortcomings:

  • Lack of functions.But you can develop the extension by yourself
  • Lack of testing.There maybe some bugs,sodo not use Pudron in large projects
  • The author has limited time and can not continue to develop Pudron for the time being.
  • The author's knowledge is limited, and some algorithms may be a little slow.

I hope everybody can learn Pudron positively.After all, its potential is still great. I will try to make Pudron present the most powerful functions with the simplest sentences.

Download

Please download Pudron in Release page.

Compile

  • Windows:Use Msys2 to compile it.Remember to install gcc and make first,then execute make PLATFORM=WINDOWS
  • Linux:Install the library libiconv first,execute make PLATFORM=LINUX
