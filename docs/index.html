自我介绍

Pudron是一门易用、轻巧、扩展性强、潜力巨大的解释型编程语言。

Pudron自认为骄傲的地方:

  • 代码格式简单,没有特别复杂的语句,容易看懂,易学
  • 开发效率高,Pudron正努力把别人几百行代码缩成一行
  • 支持面向对象、面向过程等编程方法,打起代码随心所欲
  • 可扩展,目前采用动态链接库来扩展Pudron库
  • 潜力巨大!!!

Pudron也有不足:

  • 目前功能较少,不过大家可以自己扩展
  • 较少测试,可能有未知bug,所以先不要把Pudron用于大项目
  • 作者时间有限,暂时不能继续开发Pudron
  • 作者知识有限,有些算法实现可能较慢

我希望大家能积极学习使用Pudron,毕竟他的潜力还是很大的,我将致力让Pudron用最简单的语句呈现最强大的功能。

下载

在Release页选取相应的包下载。

编译

  • Windows:使用Msys2编译,记得先安装gcc和make,运行make PLATFORM=WINDOWS
  • Linux:编译前先安装libiconv库,运行make PLATFORM=LINUX

如果你是第一次接触Pudron或者第一次接触编程,可以看详细教学,如果你认为你已经有所了解,可以看简明文档。
