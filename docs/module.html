Pudron标准模块

本文档说明各Pudron标准模块的用法,在使用标准模块之前记得先加上代码import 标准模块名;。

error

本模块用于提供异常处理ID。

提供的整数型ID

  • ERROR_INDEX:索引异常,常用于subscript
  • ERROR_CALCULATION:计算异常
  • ERROR_FILE:文件操作异常
  • ERROR_ARGUMENT:传入参数异常
  • ERROR_MEMORY:内存异常

math

本模块集成了常用的数学运算函数。

函数(返回值为double,参数可以为int或double)

  • sqrt(a):计算a的开方
  • pow(a,b):计算a的b次方
  • acos(a):计算a的反余弦
  • asin(a):计算a的反正弦
  • atan(a):计算a的反正切
  • atan2(a,b):计算a/b的反正切
  • cos(a):计算a的余弦
  • cosh(a):计算a的双曲余弦
  • sin(a):计算a的正弦
  • sinh(a):计算a的双曲正弦
  • tan(a):计算a的正切
  • tanh(a):计算a的双曲正切
  • exp(a):计算e的a次幂
  • log(a):计算a的对数
  • log10(a):计算以10为底a的对数
  • ceil(a):计算大于或等于a的最小的整数值
  • fabs(a):计算a的绝对值
  • floor(a):计算小于或等于a的最大的整数值
  • fmod(a,b):计算a除以b的余数
