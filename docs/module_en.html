Pudron Standard Modules/a>

This document explains the usage of each Pudron standard module. Remember to add the code import standard module name; before using the standard module.

error

This module is used to provide exception handling ID.

Integer ID provided

  • ERROR_INDEX:Index exception, commonly used in subscript
  • ERROR_CALCULATION:Calculation exception
  • ERROR_FILE:File operation exception
  • ERROR_ARGUMENT:Incoming parameter exception
  • ERROR_MEMORY:Memory exception

math

This module integrates commonly used mathematical operations functions

Functions(the return value is 'double',parameter can be 'int' or 'double')

  • sqrt(a):Calculate the root of a
  • pow(a,b):Calculate the b-th power of a
  • acos(a):Compute the arc cosine of a
  • asin(a):Calculate the arcsine of a
  • atan(a):Calculate the arctangent of a
  • atan2(a,b):Calculate the arc tangent of a /b
  • cos(a):Compute the cosine of a
  • cosh(a):Compute the hyperbolic cosine of a
  • sin(a):Calculate the sine of a
  • sinh(a):Calculate the hyperbolic sine of a
  • tan(a):Calculate the tangent of a
  • tanh(a):Calculate the hyperbolic tangent of a
  • exp(a):Compute e to the power of a
  • log(a):Calculate the logarithm of a
  • log10(a):Calculate the logarithm of base 10
  • ceil(a):Calculate the smallest integer value greater than or equal to a
  • fabs(a):Calculate the absolute value of a
  • floor(a):Calculate the largest integer value less than or equal to a
  • fmod(a,b):Calculate the remainder of a divided by b
